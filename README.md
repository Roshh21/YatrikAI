<h1>YatrikAI</h1>

<p>
A modern React + TypeScript travel web app that helps you plan trips, estimate budgets, and discover music playlists for your journey.
</p>

---

## Live Demo
Check it out here: https://yatrik-ai.vercel.app/

## Features

- **Plan Your Trip:** Organize travel plans easily.  
- **Estimate Your Budget:** Calculate approximate travel costs for flights, hotels, and activities.  
- **Music Recommendations:** Suggests playlists based on your travel mood and destination.  
- **Responsive Design:** Works perfectly on desktop and mobile devices.  

---

## Tech Stack

| Technology        | Purpose                           |
|------------------|----------------------------------|
| React             | Frontend UI library               |
| TypeScript        | Strongly typed components        |

## Getting Started (Local)

Clone the repo:
```bash
git clone https://github.com/Roshh21/YatrikAI.git
```

Install dependencies:
```
pnpm install
```
Run the development server:
```
pnpm dev
```
